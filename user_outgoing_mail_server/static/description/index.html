Outgoing Mail Server Per User

User Settings

In User, mark this boolean "Use own Mail Server" to enable facility of using outgoing mail server for this user

Outgoing Mail servers

Go to Settings > Technical > Outgoing Mail Servers

Outgoing Mail servers Settings

Set "Related User" for which you are setting this Outgoing Mail Server and you are all set to use this feature.

Output

See in mail that User's own mail server is used each time he sends an email
